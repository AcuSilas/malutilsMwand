# malutilsMwand

<!-- badges: start -->

<!-- badges: end -->

The goal of malutilsMwand is to provide both day-to-day basis utility functions
as well as personal growth helper functions

## Installation

You can install the development version of malutilsMwand like so:

``` r
remotes::install_github("AcuSilas/malutilsMwand")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(malutilsMwand)
## Get rewarded for working out 
workout_reward(20, c(pushups = 20, squats = 30, burpees = 10))
```
